# Proyecto 2 - Etapa Final

Este proyecto forma parte del curso de desarrollo web de TripleTen.

## Estructura del proyecto
- `index.html`: estructura principal de la página.
- `/styles/style.css`: contiene los estilos principales.
- `/vendor/normalize.css`: restablece los estilos por defecto del navegador.
- `/images/`: carpeta para almacenar las imágenes del sitio.
- `.editorconfig` y `.prettierignore`: archivos de configuración de estilo y formato.

## Descripción
Página web de la **Biblioteca Triple Peaks**, con una sección de libros recomendados ("Los preferidos de nuestro staff"), encabezado, navegación y diseño basado en Flexbox.
